# 🔖 프로젝트 소개
잊자(_ITZZA_)는 일기장으로 이별 후 나의 마음을 돌아보고, <br>
커뮤니티에서 닉네임으로 소통하고 정보를 공유하며, <br>
레포트를 통해 주별 기록을 확인하는 앱 입니다.
